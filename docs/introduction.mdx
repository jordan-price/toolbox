# Toolbox PHP Package

Toolbox is a comprehensive PHP package that provides a collection of useful tools for common development tasks. It includes tools for weather data, cryptocurrency operations, email handling, Eloquent database interactions, and time management.

## Features

- **Weather Tool**: Fetch and process weather data
- **Crypto Tool**: Handle cryptocurrency-related operations
- **Email Tool**: Simplified email sending and management
- **Eloquent Tool**: Enhanced Eloquent ORM utilities
- **Time Tool**: Time manipulation and formatting utilities
